# To Lower Case
Implement function `toLowerCase()` that takes in a string parameter `str` and returns the same string in lowercase. Do not modify the original string.

### Example 1:
```
Input: "Hello"
Output: "hello"
```

### Example 2:
```
Input: "here"
Output: "here"
```

### Example 3:
```
Input: "LOVELY"
Output: "lovely"
```
